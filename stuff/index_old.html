Welcome to the gallery!

Here you'll be able to find the full-sized versions of any images I've put up here, as well as a little description thing about them.

Neon

A quick doodle using a black canvas because the idea felt quite cool, and more neon colours felt fun to just mess about with.

Bird

If you can't tell, this is supposed to be a bird I guess. This started in a bored doodle of a more cartoonish bird, but I decided to add some colour and kinda just wanted to throw some paint around, which I ended up doing in a more literal sense. I've never done things in this sort of style before and I've got to say, I rather like it.

Bloopyfloop

A quick doodle for CoMC, I did this in the same style as the other CoMC thing, because the style's great for speed. And no, the name is beautiful.

Life in a Felled Tree

THIS, oh my goodness gracious me, this blooming drawing. It took so much out of me because it took so long to do, plus as time went on, the fear of smudging it went exponentially up, and there are still visible smudge marks on it that I couldn't get rid of. At least it's sort of passable in looks, despite the dodgy scan making the contrast go all funny.

Lobalug

An attempt at a more cartoonish digital painting I made for a CoMC task, but also took as a personal challenge and chance to try out a style that I now love for anything I don't want to spend an age on.

Paper Sky

Made this for the cover of a spotify playlist, not quite sure why I had the idea honestly, so I suppose it's up to you to interpret it as you wish.

Shading Practice

After not really doing much arty stuff for a bit, I went back to some exercises, and this is the one I did for shading spheres using multiple colours for the light.

Untitled

This attempt at abstract art of sorts came about because I hadn't much inspiration, and just kind of scribbled all over the page in frustration.

Eye

I originally made this for a spotify playlist cover, but then switched it out for something else when it didn't quite fit.

Hair

Did this while learning how to draw hair digitally using an online tutorial to get the technique. At first, this was just some hair, but after deciding that the faceless demon I had created was pure nightmare fuel, I quickly whipped up an admitttedly badly shaded face.

Walking

Done just after the my first digital thing, this is just a digitally coloured version of a pencil sketch I did when bored. I wasn't too great with the hand-eye coordination with the drawing tablet, so instead of practising with drawing exercises I chose to trace an old drawing, since that would be more fun.

Person at Night

This is actually the first thing I did digitally! I copied a different sketch I'd done with pencils, as a reference, and after a stupidly long time, I ended up with this.
